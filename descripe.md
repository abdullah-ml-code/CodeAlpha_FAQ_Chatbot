#  AI-Powered FAQ Chatbot
A smart chatbot designed to answer Frequently Asked Questions (FAQs) using Natural Language Processing (NLP). This project was developed during my **CodeAlpha Artificial Intelligence Internship**.

##  How it Works
The chatbot utilizes **TF-IDF Vectorization** and **Cosine Similarity** to understand user queries. Even if the user words their question differently, the bot can identify the closest matching answer from its knowledge base.

##  Features
- **NLP Powered:** Uses machine learning techniques to find the best answers.
- **Modern GUI:** Built with `CustomTkinter` for a professional look.
- **Efficient:** Quick response time with low resource consumption.

##  Tech Stack
- **Language:** Python 3.x
- **ML Libraries:** Scikit-learn
- **GUI Library:** CustomTkinter
